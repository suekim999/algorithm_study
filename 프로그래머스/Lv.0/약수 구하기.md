## 문제 설명
정수 n이 매개변수로 주어질 때, n의 약수를 오름차순으로 담은 배열을 return하도록 solution 함수를 완성해주세요.
<br>

## 문제 풀이

```swift

import Foundation

func solution(_ n:Int) -> [Int] {
    var result = [Int]()
    
    for num in 1...n {
        if n % num == 0 {
            result.append(num)
        }
    }
    result.sort()
    
    return result
}
```

```swift

import Foundation

func solution(_ n:Int) -> [Int] {
    return (1...n).filter { n % $0 == 0}.sorted()
}
```
