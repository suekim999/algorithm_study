## 문제 설명
 별의 높이와 너비를 1이라고 했을 때, "*"을 이용해 직각 이등변 삼각형을 그리려고합니다. 정수 n 이 주어지면 높이와 너비가 n 인 직각 이등변 삼각형을 출력하도록 코드를 작성해보세요.
<br>

## 문제 풀이

```swift

import Foundation

let n = readLine()!.components(separatedBy: [" "]).map { Int($0)! }

for i in 1...n[0]{
    for j in 1...i {
        print("*", terminator: "")
    }
    print("")
}
```
