## 다음의 규칙을 지켜 내용을 만들어주세요
- sorted() : 정렬된 값을 돌려줌

  ```swift 
  var new = array.sorted()
  ``` 
  array는 실제 정렬되지 않음.

- sort() : 정렬
  ```swift 
    var new = array.sorted()
  ``` 
  array가 정렬됨

- String(repeating: String, count: Int) : repeating 문자를 count만큼 반복한 결과를 반환

  ```swift 
      var repeatedLetter = String(repeating: "hi", count: 3)
      //hihihi
  ```
